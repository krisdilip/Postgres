### Connect to Database using pgAdmin
1. Switch to "pgAdmin 4" "bin" directory 
   
   cd "D:\Project\pgsql\pgAdmin 4\bin"

2. Start pgAdmin4  
   
   pgAdmin4.exe

3. Master Password
   
   Set the "Master Password" when prompted.

4. Register - Server
   
    HostName/address =  127.0.0.1

    Port = 5432

    Maintenance Database = postgres
    
    Username = postgres
